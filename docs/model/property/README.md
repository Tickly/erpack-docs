# 属性

## Property

| 参数     | 说明                                                                                                     |
| -------- | -------------------------------------------------------------------------------------------------------- |
| label    | 字段文本，可以在很多地方自动引用。比如表单的 label，表单验证时的 message，表格的列标题，描述组件的 label |
| required | 是否必填                                                                                                 |

## StringProperty

继承自`Property`

| 参数 | 说明     |
| ---- | -------- |
| min  | 最小长度 |
| max  | 最大长度 |

## NumberProperty

继承自`Property`

| 参数 | 说明                                           |
| ---- | ---------------------------------------------- |
| min  | 最小值，验证不通过会提示`{label}不能比{min}小` |
| max  | 最大值，验证不通过会提示`{label}不能比{max}大` |
