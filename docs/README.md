# erpack

一个基于`ant-design-vue@1.x` `vue@2.x`进行二次封装的组件库，使之更加好用。

## 安装

`yarn add erpack`

## 使用

```js
import Vue from 'vue';
import antd from 'ant-design-vue';
import 'ant-design-vue/dist/antd.css';
import erpack from 'erpack';
import 'erpack/dist/erpack.css';
// 先装ant-design-vue
Vue.use(antd);
// 再装erpack
Vue.use(erpack);
```
