# 封装一个搜索组件

建议封装一个搜索组件，用来统一整个系统的交互。

比如这个搜索组件可能包含【搜索】【重置】【展开/收起】。这样在写页面的时候，页面只需要提供给搜索组件，我需要哪些搜索条件即可，而无需关心一些与页面无关的逻辑。
